Orc Blah

Here are some special common orc words that you should know. Most other words we use are simply mispellings of English words.
  • Lat = You
  • Yub = Yes
  • Nub = No
  • Gruk = Understand
  • Ug = Hail
  • Gug'ye = Goodbye
  • Thanks = Rulg
  • Rukh = Army
  • Tribe = Collection of orcs
  • Hukor = Legendary orc
  • Krath = Gods
  • Kuul = Servants
  • Goth = Master
  • Mog = Voice
  • Lusk = Axe
  • Zult = Scimitar
  • Womp = Mace
  • Kigg = Spear
  • Igg = Dagger
  • Oligr = Bow
  • Shinies = Gold
  • Ash = One
  • Dub = Two
  • Gahk = Three
  • Futh = four
  • H = Five
  • Gruntee = New member of shadowclan
  • Grunt = Older member of shadowclan (full member)
  • Mojokii = Magic using gruntee
  • Mojoka = Magic using grunt
  • Magrukii = Necromancy using gruntee
  • Magruka = Necromancy using grunt
  • Elder = Founder of Shadowclan
  • Makr = Orc that makes things
  • Luskr = Orc that uses axe
  • Me = I (orcs never refer to themselves as "I")
